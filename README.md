# Food Safety Analytics Dashboard

A full-stack application that combines food delivery order data with NYC Restaurant Inspection Results to uncover financial risks tied to food safety. The dashboard analyzes rodent violations, revenue at risk, health grades, and provides a watchlist of high-risk restaurants.

## Deployment

The application is deployed and accessible at: [https://rat-challenge.vercel.app](https://rat-challenge.vercel.app)

To use the app, simply open the link above in your browser.

## Notes

The PDF download feature requires browser permissions to be enabled for downloads from Vercel. If the download doesn't work, check your browser settings to allow downloads from the site.
